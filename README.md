# Project Management Backend API

A RESTful backend API for managing users, projects, and daily progress reports (DPR).
This system demonstrates core backend development concepts including authentication, role-based access control, and relational database design.

## Tech Stack

* Python
* FastAPI
* MySQL
* SQLAlchemy ORM
* JWT Authentication
* Postman (API Testing)

## Features

* User registration and login
* Role-based access (Admin, Manager, Worker)
* Project creation and management
* Daily Progress Report (DPR) tracking
* Secure authentication using JWT
* RESTful API architecture
